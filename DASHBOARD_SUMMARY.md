### 🚀 **How to Use**

#### **Quick Start**
```bash
# 1. Install dependencies
cd /Users/udishkolnik/3/D.E.L.T.A
npm install

# 2. Start backend services (in separate terminals)
cd backend && python real_api_server.py
cd BOT && python bot_server.py

# 3. Start dashboard
npm run dev

# 4. Open http://localhost:3000
```
